Long-Scrolling Website

In this project, I created a long-scrolling website to express the history of my game's past across time. The timeline starts in the present day, when the territory is entirely owned by one faction. The user can scroll downward to see the history of how this took place. The site displays the year the user has scrolled to, along with the current era and some explanation of the event which led the map to this point.

CSS Animations

These simple CSS animations are a small handful of experiments I conducted, testing various features of CSS. In the first, I tested event-based transform-based animation for buttons. In the second, I tested looping keyframe animation for load screens. In the third, I tested non-looping keyframe animation for pop-out text.

Website Redesign

Together with Chao Wu and Yuki Tan, we redesigned the application process of the CSU East Bay Multimedia Masters program website. I focused primarily on the Apply page, formatting standardization, and CSS.
